5. 데이터 타입이 사용 가능한 문법 위치
	1. 변수 타입
	2. 메소드의 반환 타입 - 리턴타입 메소드명([..]){[..]}
	3. 메소드의 argument or parameter - 리턴타입 메소드명(변수타입  변수명 [,..]){}
	4. 생성자의 argument or parameter - class명(변수타입  변수명 [,..]){}
	
6. 타입 종류
	기본 타입(8가지) / 참조 타입(모든 class를 기반으로 활용)

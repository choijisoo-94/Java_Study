5. eclipse에서 maven 사용을 위한 방법
	1. 개발자들에게 최대한의 편리성 제공
	2. 4버전 이후부터 maven 기능의 plug in 이 이미 내장
	3. 단점
		- 소소한 버그
		- jdk가 미 제공하는 library들을(*.jar) 자동 다운로드 시킴
		- 껍데기만 다운로드 시키는 경우도 있음			
	4. 실제 개발 방식
		1단계 : maven 기반의 project로 설정
		2단계 : maven 자체의 설정 파일에 필요한 library들을 설정
		3단계 : library가 원격 저장소로부터 local 저장소로 다운로드가 됨
		4단계 : library 사용해서 개발
